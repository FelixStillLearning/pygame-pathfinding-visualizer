# Pathfinding Visualizer

Visualisasi algoritma pencarian jalur (BFS, DFS, A*, Dijkstra) pada grid. 

## Struktur Folder
- `src/` : Kode sumber Python
- `assets/` : Gambar/icon
- `docs/` : Dokumentasi
- `tests/` : Unit test

## Algoritma
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- A* Search
- Dijkstra's Algorithm
